流花宫 法器

驭使花木

亦或在敌人身上种下花毒，持续吸取其生命精元。
必要之时亦可激发流淌于血液中的灵狐本能，
以灵力幻化出利爪对敌人造成疾风骤雨般的伤害。

持续治疗

异芳治疗专精，源自紫降真所撰写的关于治疗和回复生命的术法典籍 《绕香谱》，可将花草万物间的治愈之力转化为自身灵力， 持续治愈身边的伙伴，配合从十二种善花益草中提炼出的“香魄”， 造成起死回生般的奇迹疗效。

灵石防护

可召唤符文石阵，与团队共同承伤，
或将灵石之力导入刀体进行精准的格挡。

可召唤灵兽摩罗辅助治疗。

群体治疗

近身杀敌，剑式刚猛锐利，迅若飞星。
